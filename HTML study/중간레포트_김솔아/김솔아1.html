브라우저라고 불리기도 하는 웹 브라우저(Web Browser)는, 사용자에게 웹 서버 컴퓨터에 접속하고 웹 페이지, 이미지,사용자에게 웹 서버 컴퓨터에 접속하고 웹 페이지, 이미지, 동영상 음악 등 다양한 데이터 다운받아 보여주는 소프트웨어이다. 그림 1-2은 대표적인 크롬 웹 브라우저를 보여준다.

웹 프로그래밍 개요

웹의 기본 목적 웹의 기본 목적은 한 컴퓨터에서 만든 문서(dacument)를 다른 컴퓨터에서 쉽게 볼 수 있도록 하는 것이다.
왜 Web인가? 전 세계의 컴퓨터들을 인터넷으로 거미줄처럼 연결하고 웹 문서를 쉽게 주고받을 수 있도록 시스템을 만들고 WWW(World Wide Web), 간단히 줄여 웹(Web)이라고 부른다.
웹 페이지를 구성하는 3요소
  • HTML - 문서의 구조와 내용
  • CSS(Cascading Style Sheet) - 문서의 모양
  • Javascript - 행동 및 응용 프로그램
